# Consider how often the user will see the animation

An animation that feels delightful once can feel annoying on the hundredth repetition. Frequency matters as much as craftsmanship.

## Practical frequency guide

| Frequency | Suggested motion level |
| --- | --- |
| One-time / onboarding | expressive motion is acceptable |
| Daily | subtle and fast |
| Hourly | very subtle or nearly instant |
| Constantly | usually no animation |

Use this to decide whether motion should:

- exist at all
- be shortened heavily
- appear only on first-run or first-view
- become instant for power-user flows

Frequent controls should optimize for fluency, not performance art.
